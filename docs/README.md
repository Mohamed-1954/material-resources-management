# Dossier d'analyse et de modelisation

Projet : **systeme de gestion des ressources materielles d'une faculte**

Ce dossier contient les artefacts demandes pour le rapport et la presentation :

- `requirements/analysis-requirements.md` : introduction, synthese d'analyse, acteurs, exigences, regles, scenarios et wireframes.
- `requirements/requirements-diagram.mmd` : diagramme des exigences.
- `bpmn/*.mmd` : processus metier BPMN-like en Mermaid.
- `uml/use-cases.puml` : cas d'utilisation global et par acteur.
- `uml/package-diagram.puml` : regroupement logique en packages.
- `uml/class-diagram.puml` : modele de classes du domaine.
- `uml/sequence-diagrams.puml` : diagrammes de sequence principaux.
- `architecture/architecture-proposal.md` : proposition architecturale.
- `architecture/component-diagram.puml` : diagramme de composants.
- `architecture/deployment-diagram.puml` : diagramme de deploiement.
- `project-management/wbs.mmd` : Work Breakdown Structure.
- `project-management/gantt.mmd` : planning Gantt.
- `testing/consistency-review.md` : verification de coherence.
- `deployment/deployment-notes.md` : notes de deploiement academique.

Les sources PlantUML et Mermaid sont volontairement gardees lisibles afin d'etre faciles a adapter.
